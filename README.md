# TotorSwipe — Terms of Use

This repository hosts the **Terms of Use** for TotorSwipe, in English.

- **Live page:** https://acquims.github.io/totor-conditions/ (after enabling GitHub Pages)
- **Privacy Policy:** https://acquims.github.io/totor-privacy-/

## GitHub Pages

1. Push this repo to GitHub (`totor-conditions`).
2. In the repo: **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: **main** (or **master**), folder **/ (root)**.
5. Save. The site will be at `https://<username>.github.io/totor-conditions/`.

## Files

- `index.html` — Single-page Terms of Use (English), styled for dark theme to match the app.
